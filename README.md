# 时间管理助手

这是一个帮助用户监督和管理个人时间的跨平台 Flutter 应用程序。应用主要包含四个模块：主页（仪表盘）、任务管理、番茄钟计时器以及数据统计页面。

## 功能特性

*   **主控制面板**：查看当天的任务概览与进度。
*   **任务管理**：添加、编辑、删除任务，并标记完成状态。
*   **专注计时器**：内置番茄钟计时器，帮助保持专注。
*   **数据统计**：可视化的时间与任务完成数据分析。

## 技术栈

*   Flutter (支持 iOS, Android, Web, Desktop)
*   Dart

## 运行指南

确保您的系统已安装 Flutter SDK。
在项目根目录下，运行以下命令获取依赖：

```bash
flutter pub get
```

然后运行项目：

```bash
flutter run
```
